Talking Fingers

Talking Fingers is a smart web-based application that translates multilingual speech into Indian Sign Language (ISL). It uses Natural Language Processing (NLP) to break down spoken sentences into simpler text, applies ISL grammar rules using deep learning, and displays the output through a 3D animated signing avatar. The tool is designed to bridge communication gaps between the hearing and speech-impaired community and the rest of society.
